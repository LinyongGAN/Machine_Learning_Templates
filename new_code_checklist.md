# 新代码启用检查单

> Jun 21, 2025 version

## 启用前检查
- [ ] 检查所有依赖路径（如果在服务器上，搜索对方的用户名路径），不论暂时是否有关，不论是否有权限操作对方的空间，均注释或改写为本地路径
- [ ] 检查数据集路径

## 运行前检查
- [ ] 对于多数据集、多模型代码，每次运行前，检查使用的模型、数据集是否正确
- [ ] 每次运行前，检查tag是否修改，之前训练的的log与模型节点文件是否删除或妥善处理

## 运行时检查
- [ ] 对于多数据集、多模型代码，检查引入是否正确。应当包括名称、路径和数据量大小
- [ ] 从之前训练过程继续训练的，检查是否载入模型正确，存储名称是否修改得当（是否会覆盖之前节点）
